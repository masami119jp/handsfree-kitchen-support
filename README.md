ハンズフリー・キッチン
料理中に画面を触らなくていい、音声操作のレシピアプリです。

「次」と声をかけるだけで手順が進みます。濡れた手、粉のついた手のまま、料理を進められます。

お問い合わせ・不具合のご報告
アプリから報告する（おすすめ）
⚙️ 設定 → 💬 フィードバック → 不具合・ご要望を報告

アプリのバージョンや端末の種類が自動で添付されるため、原因の特定が早くなります。 アカウント登録は不要です。

このページから報告する
GitHubアカウントをお持ちの方は、上部の Issues タブからもご報告いただけます。

いただいたご報告には、内容を確認のうえ順次対応いたします。

よくある質問
声をかけても反応しません
アプリにマイクと音声認識の許可が与えられているかご確認ください （iPhoneの 設定 → ハンズフリー・キッチン → マイク／音声認識をオン）
画面下部が「聞いています...」の表示になっているときに話しかけてください
読み上げ中は聞き取りを止めています。マイクボタンを押すとすぐ聞き取りに切り替わります
換気扇や水音が大きいと認識しにくくなります。その場合はジェスチャー操作もお試しください
読み上げの声が出ません
⚙️ 設定 → 音声読み上げ がオンになっているかご確認ください
iPhoneの音量、消音スイッチをご確認ください
声の種類（Hattori／O-ren）が選べません
iPhone本体に音声データがダウンロードされていない可能性があります。

設定 → アクセシビリティ → 読み上げコンテンツ → 声 → 日本語 から、 使いたい声をダウンロードしてください。

ジェスチャー操作が反応しません
調理画面下部の ✋ ボタン、または「カメラ」と発声してオンにしてください
カメラの許可が必要です
画面右上のカメラ枠の中央に手をかざし、枠が緑になってから左右に払ってください
暗い場所では手を認識しにくくなります
材料の分量を変えたい
画面上部の 👥◯人分 をタップすると、1〜6人分を選べます。 選んだ人数に合わせて、材料の分量が自動で計算されます。

自分のレシピを登録したい
ホーム画面上部の ➕ マイレシピ追加 から、材料と手順を入力して登録できます。 写真も設定できます。

削除したレシピを元に戻したい
ホーム画面上部の 🗑 ゴミ箱 から復活できます。

「みんなのレシピ」の不適切な投稿を見つけました
レシピを開いて 🚩 通報 ボタンからご報告ください。 特定のユーザーの投稿を今後表示したくない場合は 🚫 ブロック をご利用ください。

確認のうえ、必要に応じて削除などの対応をいたします。

プライバシーポリシー
最終更新日：2026年7月10日

「ハンズフリー・キッチン」（以下「本アプリ」）における、利用者の情報の取り扱いについて説明します。

1. アカウント登録について
本アプリの利用にあたり、氏名・メールアドレス・電話番号などの個人情報の登録は必要ありません。

「みんなのレシピ」機能を利用する際は、利用者を識別するための匿名IDが自動的に発行されます。この匿名IDから個人を特定することはできません。

2. 端末内に保存される情報
以下の情報は、お使いの端末内にのみ保存され、外部に送信されることはありません。

追加したマイレシピ
お気に入りに登録したレシピ
ゴミ箱に入れたレシピ
ニックネーム
アプリの各種設定
アプリを削除すると、これらの情報も削除されます。

3. サーバーに送信される情報
以下の操作を行った場合にのみ、情報がサーバー（Supabase社のサービスを利用）に送信・保存されます。

操作	送信される内容
みんなのレシピへの投稿	レシピ内容、写真、ニックネーム、匿名ID
レシピへの星評価	評価値、匿名ID
コメントの投稿	コメント内容、ニックネーム、匿名ID
不適切な投稿の通報	対象のレシピ、通報理由、匿名ID
不具合・ご要望の報告	入力内容、アプリのバージョン、端末のOS種別・バージョン、匿名ID
これらの操作を行わない限り、利用者の情報がサーバーに送信されることはありません。

4. カメラの使用について
手のジェスチャーで手順を送る機能（任意・初期状態はオフ）でのみ、カメラを使用します。

撮影された映像は端末内でのみ処理され、保存も外部送信も一切行いません。 処理に使用した一時データは、処理後ただちに削除されます。

5. マイク・音声認識の使用について
音声コマンド（「次」「戻って」など）を認識するために使用します。 音声認識はiOSの標準機能を利用しており、音声データの取り扱いはAppleのプライバシーポリシーに従います。

本アプリが音声データを保存したり、独自のサーバーに送信したりすることはありません。

6. 写真ライブラリへのアクセスについて
マイレシピに写真を設定する場合にのみ使用します。 選択した写真は端末内に保存されます。「みんなのレシピ」に公開した場合に限り、その写真がサーバーに送信されます。

7. 広告について
本アプリは、Google社の広告配信サービス「Google AdMob」を利用しています。

AdMobは、利用者の興味関心に応じた広告を表示するために、広告識別子（IDFA）などの情報を利用する場合があります。

広告のパーソナライズを望まない場合は、iPhoneの 設定 → プライバシーとセキュリティ → トラッキング から、本アプリのトラッキングを許可しないよう設定できます
Google社のプライバシーポリシー：https://policies.google.com/privacy
8. 第三者への提供
法令に基づく場合を除き、利用者の情報を第三者に提供することはありません。

9. 投稿内容の削除について
「みんなのレシピ」に投稿したレシピやコメントは、ご自身でいつでも削除できます。

その他、送信済みの情報の削除をご希望の場合は、アプリ内の ⚙️ 設定 → 💬 フィードバック → 不具合・ご要望を報告 から、 その旨をお知らせください。確認のうえ対応いたします。

10. 本ポリシーの変更
本ポリシーの内容は、必要に応じて変更されることがあります。 重要な変更がある場合は、本ページにてお知らせします。
